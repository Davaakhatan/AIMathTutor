## üìπ Recording Tips

1. **Screen Resolution**: Record at 1920x1080 or higher
2. **Frame Rate**: 30fps minimum, 60fps preferred
3. **Audio**: Use external microphone if possible
4. **Browser**: Use Chrome or Edge for best compatibility
5. **Zoom**: Zoom in on important UI elements (Ctrl/Cmd + Plus)
6. **Cursor**: Enable cursor highlighting in screen recorder
7. **Pauses**: Don't rush - pause between sections for clarity
