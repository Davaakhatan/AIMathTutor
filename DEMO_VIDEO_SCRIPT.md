# Demo Video Script
## AI Math Tutor - Socratic Learning Assistant

**Duration**: ~5 minutes  
**Goal**: Demonstrate core features and Socratic dialogue

---

## Video Structure

### Introduction (0:00 - 0:30)
- Show the application homepage
- Brief introduction: "This is an AI Math Tutor that guides students through problem-solving using the Socratic method"
- Mention: "It never gives direct answers - only guides through questions"

### Section 1: Text Input (0:30 - 1:30)
- **Show**: Enter problem "2x + 5 = 13"
- **Action**: Click Submit
- **Show**: Problem parsed successfully
- **Show**: Tutor's initial question appears
- **Demonstrate**: Conversation starts with "What are we trying to find?"

### Section 2: Socratic Dialogue (1:30 - 3:00)
- **Show**: Student responds "x"
- **Show**: Tutor asks follow-up: "To get x alone, we need to undo the operations..."
- **Show**: Student responds "the +5"
- **Show**: Tutor guides: "Right! So if we're adding 5, how can we undo that?"
- **Demonstrate**: Multiple turns showing guided questioning
- **Highlight**: Tutor never gives direct answer, only guides

### Section 3: Image Upload (3:00 - 4:00)
- **Show**: Upload image feature
- **Action**: Upload a math problem image
- **Show**: Image preview
- **Show**: Problem extracted from image
- **Show**: Conversation starts with parsed problem

### Section 4: Hint Escalation (4:00 - 4:30)
- **Show**: Student getting stuck
- **Demonstrate**: Tutor escalates hints after 2+ turns
- **Show**: More specific guidance without giving answer

### Conclusion (4:30 - 5:00)
- **Show**: Final solution discovery
- **Highlight**: Key features:
  - Socratic method (no direct answers)
  - Text and image input
  - Adaptive guidance
  - Beautiful math rendering
- **Closing**: "Ready for students to discover solutions through guided learning"

---

## Key Points to Emphasize

1. **Socratic Method**: Never gives direct answers
2. **Guided Discovery**: Helps students learn by questioning
3. **Multiple Input Methods**: Text and image upload
4. **Adaptive**: Escalates hints when student is stuck
5. **Beautiful UI**: Clean, modern, responsive design

## Recording Tips

- **Screen Recording**: Use screen recording software
- **Clear Audio**: Record voiceover explaining features
- **Smooth Navigation**: Practice the flow before recording
- **Show Real Examples**: Use actual math problems
- **Highlight Key Moments**: Point out Socratic questions
- **Keep It Natural**: Let the conversation flow naturally

## Example Dialogue to Showcase

**Problem**: 2x + 5 = 13

**Conversation**:
```
Tutor: "What are we trying to find?"
Student: "x"
Tutor: "Exactly! To get x alone, we need to undo the operations. What operation is being applied to x first?"
Student: "Adding 5"
Tutor: "Right! So if we're adding 5, how can we undo that? What's the opposite of addition?"
Student: "Subtraction"
Tutor: "Perfect! So what should we do to both sides of the equation?"
```
